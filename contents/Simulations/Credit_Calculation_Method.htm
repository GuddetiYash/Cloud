Overview

 NOTE: This applies to both Simulate and Analytics Tier customers with Credit-Based Subscriptions. 

 

The Credit Calculation method for Simulation Runs uses a 5-minute interval block structure. Credits are deducted in 5-minute blocks throughout the entire Simulation, enhancing Credit utilization efficiency. This method is especially beneficial for Models with shorter execution times, allowing you to Run these Models using fewer Credits.

Use Case

Refer to the example and formula below to understand the 5- minute interval Credit Calculation Method:

 

If the Simulation Run time is less than 5 minutes on a 2 Core Machine, the system will deduct 0.17 Credits for that Simulation. Similarly, if the Simulation Run time is 10 minutes on a 2 Core Machine, the system will deduct 0.17 Credits for the first 5 minutes and continue adding for each subsequent 5-minute interval until the Simulation is completed. For instance, for a 10-minute Simulation, the system will deduct approximately 0.34 Credits (0.17 +0.17).

Credit Calculation for High Memory VMs

 NOTE: Please reach out to your Customer Success Manager to request high-memory Virtual Machines (VMs) if needed.  

 

The high-memory VMs provide more RAM on specific core configurations, offering extra power for your Simulations. This allows you to Run long and memory-intensive Models smoothly without being limited by the memory constraints of standard VMs.  

 

The high-memory VMs have a higher Credit rate than standard VMs and are available upon request. In the Run Configuration panel, they are labeled as 1.5X, 2X, 3X, and so on, allowing you to distinguish between standard VMs and high-memory VMs.

 

For example, if an 8-core Virtual Machine is labeled as 2X in the Run Configuration panel, it will consume twice the normal Credits for that Simulation. Generally, a standard 8-core Virtual Machine consumes 0.67 Credits (as shown in the table above) for the first 5 minutes of a Simulation. If it is labeled as a high-memory VM with 2X, it will consume 1.34 Credits for the first 5 minutes of the Simulation.
