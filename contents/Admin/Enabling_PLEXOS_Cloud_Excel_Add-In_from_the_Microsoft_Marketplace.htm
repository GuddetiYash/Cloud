Overview

The PLEXOS Cloud Excel Add-In enables you to seamlessly review and edit Model Input and Output Data and synchronize changes directly to the PLEXOS Cloud from within Microsoft Excel.

To ensure smooth usage of the Add-In, users in your Tenant must have access to the PLEXOS Cloud Excel Add-In through Microsoft Excel. This integration empowers users to interact with PLEXOS Cloud models using the familiar Excel interface, simplifying data review, modification, and upload workflows.

In environments where the PLEXOS Cloud Excel Add-In is not available, users may not be able to access it by default. In such cases, Microsoft 365 administrators must manually enable access to the Add-In.

This article provides step-by-step instructions for administrators to enable the PLEXOS Cloud Excel Add-In in restricted Microsoft 365 environments.

NOTE: If users in your Tenant have access to the Microsoft Marketplace, they can follow the steps outlined in the detailed PLEXOS Cloud Excel Add-In article to install the Add-in directly.

Prerequisites

Before enabling the PLEXOS Cloud Excel Add-In, ensure the following prerequisites are fulfilled to allow a smooth and secure deployment process

  1. Administrator Access: You must be a Microsoft 365 Global Administrator or have delegated permissions to manage Integrated Apps in the Microsoft 365 admin center.
  2. Centralized Deployment Support: Ensure that your organization’s Microsoft 365 environment supports the centralized deployment of Office Add-Ins. This feature is required to enable and distribute the Add-In across users in your Tenant.

Once these prerequisites are confirmed, you can enable the PLEXOS Cloud Excel Add-In in your Microsoft 365 environment.

 

Enabling the PLEXOS Cloud Excel Add-In

Follow the steps below to enable the PLEXOS Cloud Excel Add-In from the Microsoft Marketplace.

Ensure you are signed in with a Microsoft 365 account with admin rights, as these steps require administrative access.

  1. Navigate to the Microsoft 365 Admin Center using the following link: 

    https://admin.microsoft.com/Adminportal/Home?#/Settings/AddIns

    You will be redirected to the Add-ins management screen.


  2. On the Add-ins page, click Deploy Add-In.



    Deploy a new add-in dialog box appears.


  3. Click Next on the Deploy a new add-in screen.



    A new dialog box appears to choose the Add-In Deployment Method.


  4. Select Choose from the Store.


  5. In the Add-In store, type PLEXOS Cloud into the Search bar.


  6. Click Add next to the PLEXOS Cloud Add-In in the search results.



    A dialog box appears prompting you to review the license terms.
  7. Review the license terms and click Continue.


  8. You are now prompted to assign access for the Add-In. Choose who will have access to the Add-In. You can select from the following options:
    1. Everyone: Grants access to all users in your tenant. Ideal for wide-scale deployments.
    2. Specific Users/Groups: This option limits access to predefined groups or users within your Tenant, such as Distribution Lists (DLs) created by your organization’s IT team. It is best for pilot testing or phased rollout.
    3. Just Me: This option assigns the Add-In only to the admin performing the deployment. It is useful for validation or internal testing.


  9. After assigning access, click Next.



    The PLEXOS Cloud Excel Add-In will now be deployed and available to the selected users.

 

Verifying the PLEXOS Cloud Excel Add-In

After successfully deploying the PLEXOS Cloud Excel Add-In, users can verify its availability within Microsoft Excel by following the steps below:

 

  1. Open Microsoft Excel.
  2. Navigate to the Home tab on the ribbon.
  3. Click on Add-ins (or Get Add-ins, depending on your Excel version).
  4. In the Add-ins dialog, click on More Add-ins.
  5. Go to the Admin Managed tab.
  6. Locate and select PLEXOS Cloud Add-in, then click Add.
  7. The PLEXOS Cloud Excel Add-in will now appear under the Add-ins section in Excel, ready for use.
