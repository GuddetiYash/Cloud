User Settings

Introduction

The User Settings page empowers administrators to manage user access to various product features. It has been recognized as the basic security essential for using multiple product features.

View and Add New User

The User Settings grid allows tenant Admins to view and manage their tenant users' information at a centralized location. This information is divided into multiple columns such as Name, Email address, User Id, Status, and Action. Follow the steps below to access the User Settings page:

  1. Click on the Admin icon at the page's bottom left.
  2. Click on the User Settings page. You will notice the list of users and details for each under your tenant.
 

 

  1. To create a new user, click on ADD NEW USER button located in the top right.

    NOTE: When clicking the ADD NEW USERS button, users leveraging the Central License Manager (CLM) will be redirected to the CLM > Users page.
 

 
  1. Once you click on that, a window will open allowing you to enter the user's details, as shown in the image below.

 

 

  1. After entering the name and email, you need to add the license; click on the drop-down to see the license type.
 

 

  1. We provide two types of licenses to PLEXOS cloud users.

 

  • Modeler: The user with this license type will have administrator access in User management and will be able to create as well as manage users.
  • Read only: A user with this license can only view the list of users but cannot create or edit any user.  

 

8. Select the required license type.

9. Add roles from the drop-down menu - User, Admin. You can select one user type or both by checking the box for User and Admin.
 
 
  1. Now, select the status from Active and Inactive and click on Save

 

  • Active indicates that the user's email has been verified, and they are actively using PLEXOS Cloud.
     
  • Inactive states, users will be in inactive mode until made active again and they won’t be able to use PLEXOS Cloud while in inactive mode.
     
  • Email not Verified status indicates that the user has not verified their email and has never logged in to PLEXOS Cloud.

    •  


 NOTE: The user who has been marked inactive, must wait 30 days before they can reactivate their access to PLEXOS Cloud. The duration may vary depending on your environment. For further information on specific durations, please contact the Energy Exemplar Support Team or your Customer Success Manager (CSM).
 

  1. Once the user is created, you can see the list of users with all the details.
 

 

  1. You can now send over a welcome email to the user you have just created. Click on the message icon under the Action column. Once you click on that, it will ask you to confirm, click on OK.

 


NOTE: This feature won’t work if you are an SSO user or in an SSO setup.

Type of Roles in User Settings

 

User: They will have a read-only license type.
Admin: They will have a Modeller license type and administrator level of access given to any user.  They could be able to do the following :
  • Create/Edit/Delete a user.
  • Mark any user as Active or Inactive
  • Edit, Rename, and Archive any study
  • Create, edit, or delete studies and solutions 

 

How to Delete a User

  1. To delete any user, you’ll have to first inactive that user. Click on the Edit icon
 
  1. Then select inactive and click YES to confirm.

 

 

  1. Now click on DELETE. The action will delete the user successfully.

 

Sorting/ Filter

  1. To filter the users, click on horizontal bars just right from the columns.
  2. You'll get options to apply filters and sort columns.
