Enabling Pivot View

The Pivot View is a powerful feature that allows users to analyze and summarize large sets of data in a flexible and dynamic way. 


It lets you switch from a list view to a more powerful Pivot View. With Pivot Mode, you can easily analyze large amounts of data and gain insights that were previously hard to discover.

In PLEXOS Cloud, the Pivot view can be enabled using the Pivot View toggle button on Output Data Grid Page. It transforms large chunks of the dataset into meaningful insight. 

 NOTE: Ad-hoc charting and the ability to get SUM, AVG, COUNT, and other values by highlighting data is still available with Pivot View. In addition, you can also save the selected Pivot View under the Layout options.
 

To enable the Pivot View in the Grid, follow the steps below -

1. Go to a successful Solution.
2. Select the desired objects and properties from the Properties Panel and click Apply

 



3. It will populate the data in a list view.
4. Click the Enable Pivot View toggle button to enable the Pivot View as shown in the GIF below.

 


5. By default, it shows the Sum of all the fields. You can further adjust the columns and do customization as per your preference. You can adjust these settings from the Grid toolbar under Pivot Mode.




6. Under Pivot Mode, drag and drop columns into the rows and columns sections of the Pivot table, and use the values section to specify how you want to aggregate your data. For example, see the GIFs below -
 



Options for Aggregation under Values
The supported Aggregate functions here are - SUM, COUNT, AVG, MIN, MAX, FIRST and LAST
