Overview 

The Dashboard feature is a unified analytics platform that consolidates multiple visualization elements, including Grid Layouts, Chart Layouts, Solution Reports, and BI Analytics Reports, into a widget-based interface, providing a centralized view for streamlined analysis.

 

You can configure up to 10 widgets within a single dashboard, allowing you to compare and analyze Study data efficiently in one place.  

This feature improves accessibility and decision-making by keeping all relevant insights readily available, reducing the need to switch between multiple tools.

What’s Covered in the Dashboard?

The Dashboard tab consists of the following key elements:

[A] Types of Dashboards

  1. Shared Dashboards : Displays dashboards created by other users within your Tenant. This allows collaboration and insight-sharing across teams.
  2. My Dashboards: Displays dashboards that you have created, allowing you to manage and customize them according to your requirements.

 

[B] Types of Widgets

Widgets serve as visualization elements within Dashboards. They allow you to combine multiple layouts from various sources, ensuring a more structured and efficient data analysis process.

 


 

You can configure up to 10 widgets within a single Dashboard from below-mentioned Layouts or Reports :

  1. Grid Layouts: 
    • Displays tabular data created from the Grid tab within the Solutions page using the Save Layout functionality.
    • Provides a structured and comprehensive view of simulation data.
    • Allows easy sorting, filtering, and comparison of large datasets for efficient decision-making.
  2. Chart  Layouts:
    • Represents data using graphical visualizations created from the Grid tab (via Chart View) within the Solutions page using the Save Layout functionality.
    • Supports multiple chart types, including Line, Bar, Area, and Scatter Charts.
    • Provides intuitive visual insights, making complex datasets easier to interpret.
  3. Reports: 
    • Displays structured Solution Reports created from the Reports tab under the Solutions page using the Save Layout functionality.
    • The Solution Reports are built to address common analysis scenarios and are useful as is or as a starting point to build your own Reports. These reports provide structured analysis with custom columns, totals, formulas, and calculations tailored to specific Modeling scenarios.
  4. BI Analytics: 
    • Displays Business Intelligence (BI) Reports created from the BI Analytics tab under the Solutions page using the Save Layout functionality.
    • To create these layouts, you must add your Solution to BI Analytics to view your Solution in the BI Analytics tab.
    • These BI Reports are data-driven visualizations and analytical Reports that transform complex datasets into actionable insights. These reports help you interpret trends, monitor key performance indicators (KPIs), and make informed decisions based on structured and interactive data representations.

[C] Ways to Create a Dashboard

You can create a Dashboard in the following ways:

 

  • Upload JSON File: Use previously saved JSON files from layouts or reports and upload them to quickly configure a dashboard.
  • Create Dashboard from Scratch: Manually add and configure widgets by selecting saved Layouts and Reports within your Study to build a custom dashboard.

 

Creating and Publishing a Dashboard

As mentioned in the previous section, Dashboards can be created in two ways:

  1. Creatin g a Dashboard from Scratch
  2. Uploading a JSON File

[A] Creating a Dashboard from Scratch

Refer to the following steps to create a Dashboard from Scratch:

  1. Sign In to PLEXOS Cloud using your credentials.
  2. Navigate to the Solutions page.
  3. Go to the Dashboard tab.

  4. Click on the + icon and select Create Dashboard.

    NOTE: Alternatively, you can use the CREATE NEW DASHBOARD  button available in the centre of the Dashboard Interface for first-time users.


     
  5. From the Widget panel, select the desired widget type, such as Grid, Chart, Solution Reports, or BI Analytics.
  6. Drag and drop the selected Layouts or Reports onto the Dashboard interface. 

  7. Click SAVE to save the configured Dashboard.

    NOTE: Alternatively, you can use the RESET button to discard the changes and create a new Dashboard.

  8. Once the Dashboard is saved, go to My Dashboard and click on the Kebab Menu (three-dot option), and select Publish.

    Once published, the Dashboard will be available to all users within your Tenant.

[B] Uploading a JSON 

Refer to the following steps to create a Dashboard using JSON file:

  1. Sign In to PLEXOS Cloud using your credentials. 
  2. Navigate to the Solutions page. 
  3. Go to the Dashboard tab. 

  4. Click on the + icon and select Upload JSON.  

  5. Drag and drop a file or choose from your local machine to upload it. 
  6. Click CREATE. 

The Dashboard will be created using the JSON configuration.

Managing a Dashboard 

Once a Dashboard is created, it appears in the My Dashboards section. You can manage it using the following options:

  • Rename a Dashboard 
  • Delete a Dashboard 
  • Un-Publish the Dashboard 
  • Export the Dashboard in JSON or PNG format

[A] Renaming, Deleting, or Un-Publishing a Dashboard 

Refer to the following steps to Rename, Delete, or Un-Publish the Dashboard: 

  1. Navigate to My Dashboards. 
  2. Click the Kebab Menu (three-dot icon) of the desired Dashboard. 
  3. Select Rename, Delete, or Un-Publish. 

NOTE: Un-Publishing a Dashboard removes its visibility from other users within your Tenant.
