Frequently Asked Questions

  1. What is the Dashboard feature, and why is it useful?

    Answer: The Dashboard feature is a unified analytics platform that consolidates multiple visualization elements, including Grid Layouts, Chart Layouts, Solution Reports, and BI Analytics Reports, into a widget-based interface. It provides a centralized view, enabling streamlined analysis, improved accessibility, and efficient decision-making.

 

  1. How many widgets can I add to a single Dashboard?

    Answer: You can configure up to 10 widgets within a single Dashboard. These widgets allow you to combine various data visualizations in one place for a comprehensive analysis.

 

  1. What types of visualizations can I add to my Dashboard?

    Answer:     You can add the following visualization elements as widgets:
    • Grid Layouts (tabular views of solution data)
    • Chart Layouts (visual representations like Line, Bar, Area, and Scatter Charts)
    • Solution Reports (structured reports with custom calculations)
    • BI Analytics Reports (interactive dashboards and key performance indicators 

 

  1. Can I create multiple Dashboards?

    Answer: Yes, you can create multiple Dashboards based on your analysis needs. Each Dashboard can contain different widgets, focusing on specific aspects of your Study.

 

  1. What is the difference between My Dashboards and Shared Dashboards?

    Answer:
    • My Dashboards: Displays Dashboards that you have created.
    • Shared Dashboards: Displays Dashboards shared by other users within your Tenant, allowing collaboration and insight-sharing.

 

  1. How do I create a Dashboard from scratch?

    Answer: To create a Dashboard manually:
    • Sign in to PLEXOS Cloud and navigate to the Solutions page.
    • Click the Dashboard tab.
    • Click the + icon and select Create Dashboard.
    • Choose Create New Dashboard.
    • Drag and drop widgets from the panel (Grid, Chart, Solution Reports, BI Analytics).
    • Click SAVE to finalize the Dashboard. 

 

  1. Can I create a Dashboard using a JSON file?

    Answer: Yes, you can create a Dashboard by uploading a JSON file. To do so:

    • Navigate to the Dashboard tab.

    • Click the + icon and select Upload JSON.

    • Upload the JSON file from your local system.

    • Click CREATE to generate the Dashboard instantly.

 

  1. How do I publish my Dashboard so others in my Tenant can view it?

    Answer:

    • Navigate to My Dashboards.

    • Click the Kebab Menu (three-dot icon) next to the Dashboard.

    • Select Publish.

    • The Dashboard will now be visible to all users within your Tenant.

 

  1. Can I edit or modify a published Dashboard?

    Answer: Yes. You can use the EDIT option available on the Dashboard Interface to make any necessary changes.

 

  1. How do I rename, delete, or un-publish a Dashboard?

    Answer:
    • Navigate to My Dashboards.
    • Click the Kebab Menu (three-dot icon) of the Dashboard.
    • Select Rename, Delete, or Un-Publish as needed.

 

  1. How do I export a Dashboard?
     

    Answer:  You can export a Dashboard in PNG or JSON format:

    To export:

    • Navigate to the Dashboard tab.

    • Click the Export option and select the desired format.

 

  1. Can I restore a deleted Dashboard?

    Answer: No, once a Dashboard is deleted, it cannot be recovered. It is recommended to export your Dashboard in JSON format before deletion if you need a backup.

 

  1. What happens if my JSON file fails to upload while creating a Dashboard?

    Answer: Ensure the JSON file meets the following conditions:
    • The file format is correct and adheres to the PLEXOS Cloud structure.
    • The JSON file contains valid widget configurations.
    • The JSON file is not corrupted.

      If the issue persists, try re-exporting the JSON from an existing Dashboard and re-uploading it.

 

  1. How can I ensure my Dashboard remains organized and effective?


    Answer:      Follow these best practices:

    • Limit widgets to essential elements to avoid clutter.

    • Use consistent naming for Dashboards for easy identification.

    • Export your Dashboards in JSON format for backups before making significant changes.
