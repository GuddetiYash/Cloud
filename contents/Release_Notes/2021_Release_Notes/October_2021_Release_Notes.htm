Temporarily Disabled Feature

The October release continues to enhance the underlying architecture including how Cloud Bridge and PLEXOS Cloud manage changes to studies.  
To allow our teams time to fully implement related changes to PLEXOS Cloud, we disabled the Congestion Analysis (Analyze Congestion) feature temporarily. We will enable it in an upcoming release.
 

New Performance Configurations are Available for Running Simulations (Beta)

We've delivered optimized virtual machine configurations with the new PLEXOS 9 engine in PLEXOS Cloud. For many models we tested, it delivers a 15-20% performance improvement over PLEXOS 8.3. You should also see faster machine spin-up times of 20-50%.  This will reduce the waiting time for the simulation to start.

To leverage these optimized configurations, select PLEXOS 9 Performance from the engine dropdown and then any of the machines in the VM dropdown. There is also a PLEXOS 9 Standard engine that uses non-optimized virtual machines.  Let us know what you think of Performance edition and how it works with your simulations.

To learn more about PLEXOS 9 make sure to attend the webcast series. More details are available at PLEXOS9.com.


Improved Real-Time Logging on In-progress Simulations

From Solutions view, a real-time log of an in-progress simulation. The log shows the PLEXOS engine log that you would see on the PLEXOS desktop. The log is updated every 15 seconds and will show the last 100 lines of the log. 

  
 

Gurobi as the Default Solver

Gurobi is now the default solver for all simulation runs for PLEXOS engines 8.3 and up that support Gurobi. Our testing has shown for many types of models that Gurobi provides strong results and performance. Users can edit the solver in Advanced Configuration before starting the simulation run. The adjusted solver will persist for the model it was edited from.


 

Improved Setup and Solutions Performance

We worked to improve load times of many of the most used pages within the application, Setup and Solutions. For very large models, we found that load times improved 10x on these screens. This should improve the user experience and workflow.


Models Workspace UI Improvements

We continue to improve the new Models workspace by adding new features and improving useability.  There is now a duplicate option:  A popup for confirmation on delete, highlighting of cells on change and we updated some of the terminology based on customer feedback.


Cloud Bridge Improvements

We made enhancements to Cloud Bridge to improve resiliency and performance on uploads.
 

Excel Plugin Improvements

We improved performance when loading objects from PLEXOS Cloud to Excel.


API Changes

Data querying improvements. Optimized queries, added appropriate indexes in data tables. Working with larger models should be easier than before. This is evolving functionality that will improve with each new release.
