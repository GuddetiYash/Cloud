Temporarily disabled features

The August release contains significant improvements to how Cloud Bridge and PLEXOS Cloud manage changes to studies. Due to these changes, there are two features that are currently disabled.

 

  • Editing input values on an object in PLEXOS Cloud
  • Congestion analysis (Analyze Congestion)

 

To allow our teams time to fully implement related changes to PLEXOS Cloud, we disabled these features temporarily. We will enable them in an upcoming release.

 

Cloud Bridge versioning enhancements

Cloud Bridge now has versioning architecture that improves the ability to detect changes made on PLEXOS desktop and upload them to PLEXOS Cloud.

 

Cloud Bridge proxy authentication

Cloud Bridge now supports proxy authentication.

 

AI-based Simulation Configuration recommendations

During simulation configuration, PLEXOS Cloud will now recommend settings based on previous successful runs. The simulation recommendation is found in Run Configuration under Models >> Run.


 

Email preferences

You can now choose to receive email notifications when your active simulations reach certain statuses and when the publish to reports process completes for a solution.

To access Preferences, go to [User name] >> User Preferences.



 

Simulation management

To distinguish between administrative and individual user simulation management, simulation management in Administration options is now called “Admin Simulation Management”. Admin Simulation Management shows all simulations for an organization, while Simulation Management shows only a user’s own simulations.

Admin Simulation Management

[Username] >> Administration >> Admin Simulation Management

 



 

Simulation Management

We renamed "Run Management" to "Simulation Management".

 

[Username] >> Simulation Management

 


 

We added the following functionality to the Simulation Management page:

  • Refresh button – Updates the data cards and table with most recent data
  • Simulation tags column – Shows tags associated with each run to improve filtering of table data
  • Simulation messages dialog – Click to open a dialog that shows all messages logged for the simulation
  • Hide columns menu – Click to open a menu to hide/show which tags are visible
  • Current Core Usage card – Shows total cores currently in use out of the total available cores
  • Time zone support - Added Support for time zones in Run Management filters

Also, settings and preferences in Simulation Management now persist when you leave the page and return.



 

Monte Carlo Chart Analysis

Saved Filters

You can now save filters for future use in the Advanced Filters section of the Monte Carlo Chart Analysis page.

 

To get to chart analysis, select a successful Monte Carlo solution, then click the Chart Analysis icon in the top right of Insights.

 

To save a filter, on the Chart Analysis screen, open the Advanced Filters section and set your filter settings. Then click Save Filter. Your saved filters will appear under Saved Filters in the Advanced Filters section.


 

Solutions

Improved solution cards

Solution cards for in-progress simulations now automatically refresh every five seconds and show the following information:

  • Status symbol. Learn more about simulation statuses here.
  • Date and time simulation was triggered
  • Triggered by
  • Configuration settings
  • Engine
  • Solver
  • Artifacts to view and download, including solutions files and simulation logs
  • Real time engine logs


 

Output trend chart labels

Added labels to output trend charts to clearly communicate units.


 

Solution output data grid enhancements

We renamed Solution Data Grid to Solution Output Data Grid. Use the Solution Output Data Grid to view output data in a grid format. It supports copying cells and right clicking to export/copy data. You can also do a full export to csv/excel. Additionally:

 

  • We added a drop-down filter called Pivot from which you can select a List, Name, or Property
    • List – Shows a list view of selected data with the columns Date, Name, Property, Object Type, Value, Unit
    • Name – Groups selected data by object name
    • Property – Groups selected data by property
  • The Solution Output Data Grid supports unlimited objects
  • Choose up to five properties at at time to view in the grid
  • 6 hours, 6 days, 6 weeks, 6 months and 6 years worth of data can be displayed at a time


 

Map style options

On the map views in Setup and Solutions, we added a drop-down menu for selecting different map styles. Available options are Street, (default), Light, Dark, Outdoors, Satellite, and Canvas. Your last selected map style is saved so it will persist when you navigate away and return.


 

Output data in the object inspector

We added an Output Data tab in the object inspector that shows the selected object’s output properties and values. To access the object inspector, click on an object on the Insights map in Solutions. The object inspector panel will open on the right. The Output Data tab only appears when the selected object has associated solution data.



 

API enhancements

  • Simulation state and orchestration fixes
  • Newer Azure Scale Set support
  • OData performance enhancements
  • Changed Webhook Runtime Metric values from timespan formatted string to long milliseconds
  • Added new 'NewRetryId' property on Failed status webhook requests
