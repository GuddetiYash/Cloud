In-App Reporting Updates 


Configurable In-App Reporting 

 

This new and exciting feature allows you to generate your own Solution Output report in PLEXOS Cloud. You can select output properties, manipulate reported values with mathematical operators, and generate report charts to export. 

 

This update also comes with the following new EE Reports that can be used as is, or as templates to create your own customizable reports. 

  • IRP Reports
  • Generation by Type
  • Market Prices Report
  • Generation-Weighted Prices Report
  • Zonal Summary Report
  • Fuel Consumption Report
  • Energy Balance Report
  • Gas System Summary Report
  • Hydrogen Production Report
  • Gas Storage Report 

 

A sample of a report output in PLEXOS Cloud is shown below.
 

 

You can find detailed information on configuring your own reports using JSON files in the following article In-App Configurable Reporting in PLEXOS Cloud. 

 

IRP Reporting Enhancements 

Users can now report values at a Unit/Object Level. You can also specify the simulation phase for reporting a group or column. The UoM (units of measure) for each column can also be edited to report different units. 
 

Study Access and Management 

Request Access to a Study 

You can now request access to Studies within your environment. Under the Study dropdown, you can click to Request Access to any private Studies that are displayed as grayed-out as shown below. 

 

 

You will then be asked to confirm the request and click Yes. 

 

 

 

Approving/Denying Study Access via Email 

The Study Owner can then approve or decline your access request via email. The Study Owner will receive an email with information on the request in a link to Approve or Decline it. A sample of the email is shown below. 

 

 

If approved, you will have immediate access to the Study and receive an email notification. You can also open the Study in PLEXOS Cloud by clicking the View Study button from the email. 

 

 

You will also be notified via email if your request is declined. A sample email message is shown below. 
 


 

Public and Private Study Icons 

Users can now easily identify if a Study is Public or Private based on new icons to the left of the Study name in the Study dropdown. An icon representing multiple people indicates Public while an icon with multiple people and a lock represents Private. The icons are shown below. 

 

 

 

You can learn more about the ability to make a study public in the July PLEXOS Cloud Release Notes.

Object Inspector Improvements 

Objects with long names have now been truncated in the Object Inspector to ease reading. The Object Inspector is the right panel that comes up when selecting an object on the Map or from Solution Summary.

New icons are available for transmission lines with color-coded matched icons. 

 


 

You can also hover over geocoded objects, with copy support  now incorporated for the object information. 
 

 

Congestion Analysis Improvements 

It is now easier for users to gain insights on system congestion in PLEXOS Cloud. You can read more about the enhancements in this release below, understanding congestion in this article about Congestion Analysis in PLEXOS Cloud . 

 

Listing Congested Lines 

PLEXOS Cloud can now tell users which lines in the system are congested during the specified interval. 

 

For geocoded studies, you can navigate to the Map icon shown in the top righthand corner of your screen. In the property dropdown on the left side, under Transmission and Line dropdowns, select the Congestion output. 

 

 

 

On the bottom of the screen, select the Period Type, Sample, Phase, and Time Period. When Output Trend is selected, you can see the Congestion ($/MW) across the selected interval. If you click on a time on the trend chart where there is Congestion in the system, PLEXOS Cloud will tell you which lines are congested.

 

 

You can see the congested lines listed in the top right hand corner. In this example, if you select the point at 12 May 2023, you can see that Line West1_West2_115 has $8.30/MW and Line West_69_3 has $5.11/MW of congestion. To drill down further, click on the Line (shown below in red) or the listed lines. This will allow you to see the Input, Output, Connections, and Congestion on the line in your browser window. 

 

 

Viewing Congestion Analysis in a browser tab 

Users can now view Congestion Analysis reports without having to open a new browser tab. 

After clicking on a Line, go to the Congestion tab to select the Induced Flow, Congestion, or Node Flow Report. 

 



 

Minor improvements and bug fixes 

Solution Summary results now report Monthly and Yearly results correctly. Users previously experienced issues when trying to view Study results.

You can also now access the Diagnostics button on the Solutions panel for runs that are In Progress.
 

The performance for loading geocoding items on the Solutions Map for large studies including Eastern Interconnect has been improved.
 

To eliminate any truncation of tabs, the names of the following tabs within the Object Inspector under Solutions have been changed 

  • Input (replacing Input Data)
  • Output (replacing Output Data)
  • Connections (replacing Map & Bus Diagram)
  • Congestion (replacing Congestion Insights) 


The Congestion Report names have also been truncated in the Congestion tab. 

  • Induced Flow (replacing Induced Flow Decomposition)
  • Congestion Charge (replacing Congestion Charge Decomposition)
  • Node Flow (replacing Node Flow Report)
