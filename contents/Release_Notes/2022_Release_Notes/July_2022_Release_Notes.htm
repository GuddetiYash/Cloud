Solution Output Data Grid

After a successful model run, you can now view more data with faster performance across the data grid. There has also been a major update to the Solution Output Data Grid for retrieving data across dates. Learn more about the new features below: 


Updated Approach for Retrieving Data  

You can now view more data in List view without having to filter by date. By default, you can now view data for the full horizon. 
 

Previously, you could only view a certain number of days, months or years at a time, and would constantly have to change the date to view data from the desired time interval. Now, if you run a model with a 1-year horizon and intervals of 1 hour, for example, you can view output data for all the intervals in the horizon without having to filter by date. 
 

While viewing output data in List view, you click the Next button, as shown below, to load more data. 
 


 


Study Access and Management   

We have improved study access and management on PLEXOS Cloud to enhance the experience for modelers and administrators.

 

Study Access

You can choose to make studies Public as shown below. A Public Study is one that can be accessed by all users in the system. Each user’s license will determine the level of access they have to the Study. To make a Study Public, navigate to the Overview tab under the Study, and in the upper right corner click Manage Team. Under Manage Team Members, switch the toggle to make the Study Public.

 


When the option appears to approve, select Yes to confirm that the study will be moved from Private to Public.
 


 

Excel Plug-In Enhancements

The Excel Plug-in now can now support multi-banded variables, scenario data, and geocoded objects for modelling. Learn more about the enhancements below.


The Excel Plug-in will now include support for multi-banded variables to so users can both create and modify variables such as Generator Heat Rate Incr or Load Point. If banded data exists, it will show up as individual rows when the pull data command is executed. The inclusion of the banded information is automatic for relevant model properties.

 

Scenario data is also supported with this release of the Plug-In.  Any modelling data existing in a user scenario will be populated in additional rows along with the base data.  

 

Users will now be able to selectively turn on and off data pulls within the template sheet without having to delete the rows that define them. This will allow the user to restrict the amount of data each pull operation receives while preserving the template definition so it can be easily reactivated without the trouble of recreation.

 

Geospatial coordinate information can now also be retrieved. This feature can be selectively turned on and off for each table by specifying Yes/No in the "IncludeGeocode" column in the Excel Plug-In template worksheet.
 

Refer to the following article for more information on the Excel Plugin for PLEXOS Cloud.
 

Cloud Bridge Enhancements

 A new version of Cloud Bridge is now available for installation under the Tools page. Once the new version is successfully installed, users should uninstall the previous version.

  NOTE: Before you delete/uninstall the old version of Cloud Bridge, please check the version number. Make sure to delete the older version.

Read more to learn about what makes this version of Cloud Bridge different from previous releases.



Cloud Bridge SSO 

Browser-based authentication is now supported in Cloud Bridge for both Single Sign-On (SSO) and non-SSO users. This means that once you click on Sign In in Cloud Bridge, you will be redirected to your browser where you can enter your email (for SSO users) or your email and password (for non-SSO users).

If you are already authenticated for Cloud in your browser, you will simply be redirected and authenticated for Cloud Bridge without having to reenter your username.

 

MSI Installer  

This new version of Cloud Bridge also uses an MSI Installer. This benefits users since it can be managed by an IT Administrator and deployed to users through automation. It also grants IT more control over the application than previous versions, which could only be updated by desktop users. You can find more information in the following article on Browser-based Authentication for Cloud Bridge on the MSI Installer and SSO capabilities.

Additional Bug Fixes and Improvements

We have continued to fix bugs affecting user experience, stability, and performance, including fixes related to Cloud Bridge.

If you have a favorite (starred) study and then your access to that study is removed, the system removes it as a favorite. Previously, if a study was listed as a favorite, you could not remove the favorite or access the study.

When you upload a study to Cloud, if your solver is set to Gurobi or CPLEX, Cloud will not overwrite your solver setting. If the solver is not Gurobi or CPLEX, Cloud will set it by default as Gurobi.

A previous issue has been resolved so users can now export drill-down summary data in .csv/.xlsx format.

Price outputs in the Solution Summary now report correctly as the Region average in the selected interval.
