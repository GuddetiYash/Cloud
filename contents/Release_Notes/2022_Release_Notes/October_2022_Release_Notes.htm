PLEXOS Cloud Desktop Integration

PLEXOS Desktop introduces a new PLEXOS Cloud Integration in version 9.1 R03. The Cloud Integration seamlessly integrates PLEXOS Desktop and PLEXOS Cloud for key workflows and reduces the dependence on the PLEXOS Cloud Bridge application. From inside PLEXOS desktop users can now:

 

  • Save a Study to PLEXOS Cloud
  • View a Study in PLEXOS Cloud
  • View the Solution list and Download\view Solutions in PLEXOS Cloud
  • Run and Configure a Model\View Model in PLEXOS Cloud
  • Sync a Study from desktop to PLEXOS Cloud and vice versa

 

Read more about getting started with PLEXOS Desktop Cloud Integrated 9.1 R03.

 

Command Line Interface (CLI)

Users can now interact with PLEXOS Cloud and automate processes using the CLI (also called the PLEXOS Cloud Toolkit). The CLI is a modern and simple way to interact with the application without having to use an API. You can trigger commands such as uploading a Study, running a simulation, downloading output in a variety of formats, and much more.

 

Read more about getting started with the PLEXOS Cloud CLI here.

 

Monte Carlo (Auto-Sample) Improvements

Monte Carlo Simulations have been revamped in PLEXOS Cloud. PLEXOS Cloud also now supports user-defined (exogenous) samples in addition to endogenously defined samples.

 

Previously, users had to populate the Stochastic Object with the total number of samples divided by the number of virtual machines (VMs). For example, if you wanted to run 100 samples over 10 machines (10 samples on each machine), the Variable Sample Draws had to be populated with the number 10 instead of 100. Now, users need only define the number of samples in the Stochastic Object. This means that if you have 100 samples, you specify 100 Variable Sample Draws. 

 

You can learn more in this article about Monte Carlo (Auto-Sample).

 

PLEXOS Cloud MS Excel Add-In

The new Microsoft Excel Add-in allows you to review and edit model input data and synchronize changes to PLEXOS Cloud directly from Excel. The Microsoft Excel Add-In replaces the now deprecated Excel Plugin. The Add-in is lightweight and supports both model data and the data files associated with the Study. Small, targeted changes or complex bulk changes based on formulas can be done easily and saved back to the cloud with just a few clicks.

 

Read more about using the Excel Add-In to enhance your modeling experience with PLEXOS Cloud.

Solution Output Grid Changes

The Solutions Output Grid has increased the yearly period type data reporting from 5 years to 30 years. You can view this change if you navigate to Solutions and then select the Grid icon.

 

Search Models and Solutions within the Solutions Panel

In the Solutions pane, a new Filter has been added that allows you to easily find the Solution or Model you are looking for. You can filter by Model name, triggered by user, status, and date range. You can also sort by Date and Model Name (A to Z).
 

Go to Solutions > Click on the Filter icon > Click More Options.

 


 

Search Models

Click in the Search Bar and enter a few keywords of your Model name. A list of models matching your search keywords will display. You can select as many models as you want. Only the latest selected Model name is displayed inside a chip, previously selected items are present inside a popup that is opened by clicking ‘+N More’ button next to the latest selected chip. Search text can be cleared by clicking on the ‘x’ icon.


  NOTE: The Search Model will only show a list of Models available for your Study.

 

 


Sort By
Sort your simulations by date or model name in alphabetical order. In the search results, use an up and down arrow (next to the Solutions label) to view results in Ascending or Descending order.

 

 


Date Range
Manage your search results by applying a specific date range. Simulations which are part your selected date range are displayed.

 

 


Search Users
This Search Bar allows you to select and view the user-specific simulations.

Status
It displays all the statuses applicable for listed simulations under models in your study. You can select any status to view the simulations under models that matches the selected status.

 

Tenant Preference Enhancements

The Tenant Preferences page allows the admin to change settings/preferences that impact what users see in the tenant. The admin can view all engines available for the tenant and decide which to show or hide to users. Additionally, the admin can replace the PLEXOS Cloud logo at the top left of the interface with a company logo.
 

 

Learn more on the Tenant Preferences page.


Study Management Enhancements

Bulk Actions

Bulk actions are now available to Archive, Delete, or Change Owners for studies in the Study Management tab under Administration.
 

 

When you click the Archive button , all the selected studies will be archived when you click Confirm.
 


Studies can only be deleted after being archived. After you have archived the selected studies you can Delete them by clicking the icon. You will then be prompted to Confirm and delete the studies.
 

 

Admins can also update the ownership of multiple studies at once. After selecting Change Owner, you will be shown a dropdown menu from which you can set the new owner for the selected studies.

 


 

 

Viewing Simulation Run Times

You can now view the Engine Run Time for each Study on the Simulation Management page.
 


Enhancements to In-App Reports (Solution Reports)

There are some minor enhancements performed on the IRP Reports section. Details are as below:

 

  1. In the Solutions tab, the header name has changed.  Analysis for [Solution _Name] replaces Insights for [Solution_Name].

 

 

  1. In the Reports tab under Solutions, the table now shows Total at the bottom of the Report which replaces Totals.

 

 

  1. When you right-click on any Report there is a new View Details option. This shows the Period type ID and Phase ID and all properties on the report. This is helpful to understand what data will show in the report and what data you have or don't have in this Solution.  

 


  NOTE: The Period Type ID and Phase ID values must be added in the JSON configuration file before uploading to PLEXOS Cloud. You will receive an error if they are not populated.

  • Red Text- properties that are not part of the Solution output data. The columns or formulas that leverage these properties will be blank or not calculate correctly. To include these on future runs for your Models, add them in PLEXOS desktop in Reports.
     
  • Black Text- properties that are part of the Solution output for this Model and therefore will populate for this Report.

 

Minor Improvements and Bug Fixes

Users can now download and initialize studies created externally stored in zipped file sizes larger than 2GB in Cloud Bridge.
