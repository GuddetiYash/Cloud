Models overview

The Models Page in PLEXOS Cloud provides tools to run, configure, and optimize models efficiently. Users can run models using various Run Types, such as Single Run, Distributed Run, and Tuning Run, each designed for different computational needs. The Run Configuration Panel allows customization of model execution by selecting the Engine Version, VM Type, and additional settings to optimize performance. Users can configure and execute models precisely through features like Solution Chaining, Simulation Groups, and Resource Monitoring, making the platform adaptable to various energy modeling requirements.
