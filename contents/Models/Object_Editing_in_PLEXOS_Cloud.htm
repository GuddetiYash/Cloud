Object Editing in PLEXOS Cloud

Locate an Object on the Map

  1. Select a study from the Study drop-down menu.
  2. Under Solutions, choose the completed solution for the model you want to edit data for.
  3. Expand the Insights Properties menu.
  4. Expand a section to show the input data you want to see on the map.

     Note: Once input data is enabled on the map, you can use Power Search to find specific objects. To do so:

    1. Click on the magnifying glass next to Properties.
    2. In the search field, type the name of an object or define a parameter.
    3. Press Enter on your keyboard. The map will now show only objects that are enabled in the properties menu and meet the criteria of your search. The result count appears below the search bar on the left.
  5. Click on a map object. The Input Data panel will open.

 

Edit Existing Properties

To edit existing properties:

  1. In the Input Data panel, double click on a property row. Items that can be changed will become editable fields.

    Editable fields include:

    • Property type
    • Value
    • Scenario
    • Band
  2. Edit the data.
  3. When you have finished updating the data, click anywhere outside of the editable fields, then click Apply.

The input data for the model now reflects the changes you made. The next time you run a simulation for the model, it will use the updated object data.
 

Add New Properties

To add a new property:

  1. In the Input Data section for a selected object, click the plus in the Property Type column.
  2. Select a property type from the drop-down menu and then set its value, scenario, and band.
  3. Click anywhere outside of the editable fields, then click Apply.
