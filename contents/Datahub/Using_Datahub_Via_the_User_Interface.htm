1. Overview

The Datahub UI provides an organized and user-friendly experience for managing your files within PLEXOS Cloud. It supports essential file operations and helps you structure your data in a way that fits your modeling needs.

 

2. How to Access the Datahub

[A] From PLEXOS Cloud

The Datahub button is located at the top-right corner of the PLEXOS Cloud interface, under the Products and Utilities icon. It provides convenient one-click access to manage your tenant’s files in a centralized and secure environment.

3. Available Functionalities in the Datahub UI

  1. Create Folders: Use the New Folder button to organize your files with folder structures.

     NOTE: When creating folders/files in the Datahub via the UI or CLI, ensure the folder names do not include special characters such as | * < > / |  ? : and do not begin with an exclamation mark (!), as these are restricted to maintain consistency and prevent pathing issues.

  2. Upload Files or Folders: Upload files directly into Datahub using the Upload functionality. Currently, you can upload one file at a time with a maximum file size of 800 MB, ensuring secure and efficient data transfer to your tenant's storage.

  3. Search Files or Folders: Use the Search Bar at the top to quickly locate specific files or folders by name, helping you navigate large file structures with ease.

  4. Delete Files or Folders: Select one or more files/folders and click the Delete icon to remove them.

  5. Download Files: Easily download one or multiple files using the Download button in the top-right toolbar.

  6. Copy File Path: Copy the relative path of a selected file for use in CLI references or for sharing within your team.

  7. Switch Views: Toggle between Grid or Tile View to suit your preference for navigating content.

  8. Refresh File List: Use the Refresh icon to instantly update the view and reflect new additions or changes made to your tenant storage.
