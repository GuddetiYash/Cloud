Viewing the Chart Analysis Dashboard

To view chart analysis, you must first run a Monte Carlo model using the Distributed setting in run configuration.

Run a Monte Carlo Model

  1. Go to Models.
  2. Select the desired Monte Carlo model.
  3. Click Run.
  4. Configure run settings and select the Distributed option. 
  5. In the Distribution Method, select Monte Carlo Samples.

     NOTE: See the Distributed Runs article for more information on how the sample amount is allocated to the virtual machines.

  6. Run the model.
  7. The solution will be queued on the Solutions page.

 

Generate Summary CSV Data

  1. Go to Solutions.
  2. Find the solution and click on it to reveal more actions.

     NOTE: The solution must have a status of ‘Completed Successfully’ before you can complete the following steps.

  3. Click the Published to Reports icon .
  4. Locate the ‘Generate Summary CSV Data for Analysis’ pipeline and click the play icon in the Action column.

     NOTE: Data automation is complete with the Pipeline Run Status is ‘Succeeded’ in the Data Automation window.

 

View Chart Analysis

  1. Wait for data automation to complete.
  2. Make sure the solution you generated summary CSV data for is selected in the solutions list.
  3. In the Insights pane, click the chart analysis icon above the map.
