Overview

The Setup tab provides a powerful interface for managing and visualizing geocoded and non-geocoded objects. With the geocoded data, you can display objects on a map, enabling you to visualize outputs and gain valuable insights into spatial relationships and memberships. 
 

Setup tab offers several key features to enhance your setup experience: 
 

  • Viewing Geocoded and Non-Geocoded Objects: Allows you to easily select geocoded and non-geocoded objects to analyze their spatial distribution or update their coordinates for accurate placement on the map. 

  • Visualize and Modify Coordinates: Easily adjust the latitude and longitude of objects using a drag-and-drop functionality, ensuring precise placement on the map. 

  • Edit and Draw Polygons: Create and modify polygon shapes to define areas or boundaries according to your specific needs. 

  • Customize Map Icons: Personalize icons for different classes and categories within the map, improving clarity and organization for better data representation. 

  • Downloading and Uploading Geocodes: Easily download geocodes in CSV or Shapefile format and upload updated geocodes back to the system. 

    • CSV File Content: Includes object names, latitude, and longitude coordinates. 
    • Shapefile Content: Contains detailed polygon data for visualizing boundaries or spatial objects.

  • Map Settings: Select from the various map types such as Street, Light, Dark, Canvas, and Satellite views with additional map details that provide better data visualization. 

 

The Setup tab consolidates these tools and provides a streamlined approach to managing spatial data, making it an essential feature for accurate visualization, customization, and efficient data handling.
