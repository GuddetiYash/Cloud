Overview

In the Output Data Grid tab, users can view and analyze Simulation results presented in a Grid and Chart format. This Grid tab displays various output data related to power system Simulations, such as generation, load, prices, or other relevant parameters. 

To view the output data in Grid and Chart views, users must select the desired Objects and Properties from the Properties panel.

For example, 

Grid View

Grid View presents a fundamental yet powerful tool for solution data analysis, offering a structured and comprehensive tabular representation of data. Its value lies in its ability to succinctly organize and display large datasets, facilitating easy comparison, sorting, and filtering of information.  

 

 

[A] Output Settings

The Output Settings drop-down allows you to control how data is displayed within the Grid tab. 

 

These settings help refine your view by determining the simulation phase, time granularity, duration, and preferred view format.

 

 

The following components are available in the Output Settings panel:

 

  1. Phase: Defines the simulation phase (e.g., ST, MT). For the Grid tab, this is automatically set to the ST (Short-Term) phase.

  2. Period: Specifies the time aggregation level, such as Year, Month, or Interval. The Grid tab uses the Interval period by default to allow detailed, time-step-based Grid examination.

  3. From – To: Enables you to define the time range for your analysis. Select the specific start and end intervals to narrow your focus on the desired portion of the simulation timeline.

  4. Compare Solutions: The  Compare Solutions dropdown allows you to compare up to four Solutions side-by-side without navigating to the dedicated Comparison page.

    To compare Solutions, select your desired Solutions (up to four) and click APPLY.

    The tab updates with additional columns, displaying side-by-side comparisons for the selected Solutions.

[B]  Show More Data

Users can click the Show More Data button to load the next batch of data if the total results are more than 100k. The Show More Data button can load more data if the current batch is loaded. After you click on Show More Data, you will see a Loading icon at the bottom. After all records are loaded, the Show More Data button is greyed out and you will be able to sort and filter all data. 

 

[C] Number of Shown Records

In the footer, you can view the text showing how much data is currently loaded in the grid versus the total number or records the query returned. If not, all records are loaded you can click the Show More Data button. 

 

[D] Select Cells and View MIN, MAX,  AVG, and more summary data

You can multi-select cells in the Grid to get COUNT, SUM, MIN, MAX and AVG for the selected records. This will be real time and the calculation changes when you select or deselect cells. 

This option is available with the Name and Property views as well. 

Chart View

The Chart view provides a visual representation of data, allowing users to analyze the Solution output through various forms such as Line, Bar, Area, and Scatter Charts. With the Chart view, analyzing Solutions for complex or large datasets is easy, so that users can make informed decisions in real-time. 

The Chart view provides extensive customization options for analyzing data more conveniently. Users can easily rename the X and Y Axes, enable or disable the Secondary Axis, incorporate shapes onto the Chart, implement Zooming and Autoscaling functionalities, Export Charts, reposition the legend and more.  


 
By default, the Secondary Axis toggle will be in the disabled state. Users can enable the toggle to view Chart Type 2 and the Secondary Y Axis. 

[A] Chart Settings

The Chart Settings allow users to select desired Primary and Secondary Y-axis properties, various Chart Types, Font Size, Markers, Legends, and Grid Lines to customize the chart according to their needs. 

  1. Primary Y Axis: This option displays the available properties for selection on the Primary Y Axis, located on the left side of the Chart. By default, the first property of the Primary Y Axis is selected for the Chart.

  2. Chart Type 1: This section showcases the available Chart types, such as Lines, Bar, Area, and Scatter, for Chart type 1. By default, the chart type is set to Line, but users can change the chart type according to their needs. 

  3. Secondary Y Axis: Users can select properties for the Secondary Y Axis, located on the right side of the Chart. By default, the last property of the Secondary Y Axis is selected for the chart.  

  4. Chart Type 2: Similar to Chart Type 1, this section presents a variety of Chart types, including Lines, Bar, Area, and Scatter, for Chart type 2. By default, the Chart type is set to Line, but users can change the Chart type according to their needs. 

  5. Data Type: It includes a Time Series and Duration Curve data types. These data types are crucial in charting for providing insights into time-dependent behavior and the distribution of values, enabling better decision-making and strategic planning.

  6. Font Size: This section provides options for adjusting the font size used for Chart context, offering choices such as Small, Medium, and Large. By default, the font size is set to Medium, but users can change the font type according to their needs. 

  7. Secondary Axis Toggle: This option allows users to enable or disable the Secondary Y axis on the chart, providing flexibility in visualization. 

  8. Show Markers: Users can utilize this option to visually display individual data points as markers on the chart, aiding in data interpretation and analysis. 

  9. Show Grid Lines: Grid lines provide users with a reference framework for interpreting the data plotted on the chart. By displaying grid lines along the axes, users can accurately gauge the scale and relative positioning of data points within the chart. This allows users to make more precise measurements, comparisons, and estimations based on the chart's visual representation. 

  10. Show Legends: This option allows users to selectively show or hide legends on the chart based on their preferences.


[B] Legends 

Legends provide users with guidance in interpreting the meaning of different elements in a chart, such as colors, object names, and property names, depending on the selected Primary and Secondary Y Axis.

 

 

  1. Users can drag and reposition the legend within the chart area according to their preference. This functionality proves useful when dealing with complex charts featuring multiple data series or categories. Users can adjust the legend's placement to avoid overlapping with other chart elements or to optimize the chart's layout for better readability. 

  2. To view a particular legend, click on the corresponding color. 


[C] Chart Title

The chart title reflects the selected period type from the Output Settings and the chosen Primary Axis properties from the Chart Settings. For example, when "Generation" is selected as the Primary Axis property and "Interval" as the period type, the chart title displays "Generation by Interval."

This feature provides immediate context to charts, helping users quickly understand the data for better analysis.


[D] Chart Types

Chart types are particularly useful for visualizing how different data segments contribute to a total, making it easier to track changes and identify patterns over time. The available chart types include:

 

  • Line Chart: A line chart displays data points connected by straight lines, ideal for showing trends over time or continuous data. This chart type helps you easily identify patterns or changes in data over a specific period.

  • LineStacked: A stacked line chart displays multiple data series as lines, with each series stacked on top of the previous one.

  • Bar Chart: A bar chart represents data with rectangular bars, where the length of each bar corresponds to the value it represents. This chart type is excellent for comparing individual data points across different categories.

  • BarStacked: A stacked bar chart represents data with rectangular bars divided into segments. Each segment represents a different data series, stacked one on top of the other within the same bar.

  • Area Chart: An area chart is similar to a line chart, but the area beneath the line is filled in. This helps emphasize the magnitude of data and is useful when you want to visualize both trends and cumulative values over time.

  • AreaStacked: A stacked area chart is similar to a line chart but with the area below each line filled in and stacked on top of the other series.

  • Step: A step chart is a variation of a line chart where the data points are connected by a series of steps rather than straight lines. Step charts are ideal for showing changes that occur at specific points in time, such as changes in pricing, inventory levels, or interest rates.


  • StepArea: A step area chart combines the features of a step chart and an area chart. Like a step chart, it connects data points with steps rather than straight lines. The area between the steps and the x-axis is filled in, emphasizing the magnitude of the changes. The filled areas help you quickly grasp the overall volume or magnitude of the data, making it easier to see how changes accumulate over a period.


  • Scatter Chart: A scatter chart plots individual data points based on two variables, displaying them as dots. It’s especially useful for identifying relationships or correlations between variables, trends, and outliers.

[E] Data Types

 Data Type includes a Time Series and Duration Curve data types. These data types are crucial in charting for providing insights into time-dependent behavior and the distribution of values, enabling better decision-making and strategic planning.

 

  • Time Series: Time series data consists of sequences of data points collected or recorded at specific time intervals, such as hourly, daily, weekly, or monthly. It is essential for tracking trends, patterns, and behaviors over time. It helps customers analyze how a variable changes over time, enabling them to forecast future trends, identify seasonal patterns, and detect anomalies.

  • Duration Curve: In the Duration Curve, the data points are sorted in descending order of magnitude, and the curve shows the percentage of time that the variable exceeds a certain level. Duration curves are particularly useful for assessing the reliability or availability of resources. For example, in energy management, a duration curve can show the percentage of time a power plant operates at a certain capacity. This helps you understand the distribution and likelihood of different levels of performance or resource availability, aiding in resource planning and risk management.  

[F] Tooltips 

Tooltips are interactive pop-up boxes that provide additional information about specific data points when users hover the cursor over them. Tooltips displays information for the X axis, Y axis, and Values, allowing users to gain insights or context without cluttering the chart with excessive labels or annotations.  

Users can leverage tooltips to quickly identify data points of interest, compare values across different data series, or understand the significance of particular data points within the chart. 

[G] Chart Customization Options 

The Chart customization options allows users to download the Chart in PNG format, incorporate shapes onto the chart, implement zooming and autoscaling functionalities, and more. 

 

[I] Download plot as a PNG

This option allows users to download the chart in PNG format. Users can utilize this option to easily download and share their customized charts with colleagues and clients, or integrate them into presentations, reports, or documents, facilitating seamless collaboration and communication.  

Click on the Camera Icon to download the Chart in PNG format.

 

[II] Zoom Button 

This option allows users to zoom in to the desired area of the Chart. Users can utilize this option to focus on specific data subsets for deeper analysis.   

To zoom in on a particular area of the chart: 

  1. Click on the Zoom Icon and select the area you wish to zoom in on the Chart. 
  2. To return to the original view, double-click anywhere on the Chart. 

 

[III] Pan Button 

This option allows users to move or scroll across the chart horizontally or vertically.  

To pan across the chart: 

  1. Click on the Pan Icon and click and hold your mouse to explore the data on the chart. 
  2. To return to the original view, double-click anywhere on the Chart. 

 

[IV] Box Select 

Box selection allows users to select a specific range or subset of data points within the chart for further analysis of the data. This option is particularly useful when dealing with large datasets or complex visualizations where isolating specific data points or regions is necessary for deeper analysis. 

 

[V] Lasso Select 

Lasso selection is another way like box selection to select a specific range or subset of data points within the chart for further analysis of the data. Instead of drawing a box to select data points, users can use the lasso tool to draw a freehand shape around the desired data points. This option is particularly useful for selecting non-linear or irregularly distributed data points.  

Lasso select is beneficial when dealing with complex data structures, such as scatter plots with overlapping points or line charts with multiple series, where traditional box selection may not be sufficient. 

 

[VI] Various Drawing Options 

The drawing options in Chart view provide users with additional tools for interactively analyzing and exploring their data. By drawing lines, freeform shapes, or circles on the chart, users can visually identify patterns, trends, and relationships in the data and perform targeted analysis on selected subsets of data for deeper insights and understanding. 

 

  • Draw Line: This option allows users to draw a straight line on the chart, typically connecting two points of interest. Users can use this feature to visually compare the relationship between two variables or to highlight a specific trend or pattern in the data. For example, users might draw a line to represent a linear regression trend or to emphasize a particular slope or angle in the data. 
  • Draw Open Freeform: With this option, users can draw a freeform shape such as a curve or a series of connected lines on the chart. This feature is useful for outlining specific areas of interest or for highlighting irregular patterns in the data. Users might use open freeform drawing to enclose data points that belong to a particular category or to mark areas with anomalous behavior for further investigation. 
  • Draw Closed Freeform: Similar to open freeform drawing, this option allows users to draw a freeform shape on the chart, but with the ability to close the shape to create a bounded region. Users can use closed freeform drawing to segment the data into distinct clusters or to define regions for analysis, such as identifying outliers or determining areas of high density. 
  • Draw Circle: This option allows users to draw a circle on the chart, with the ability to adjust the size and position of the circle as needed. Drawing circles can be particularly useful for geographical analysis or for identifying outliers in the data. 
  • Draw Rectangle: This option allows users to draw a rectangular shape on the chart, with the ability to adjust the size and position of the rectangle as needed.

 

[VII] Erase Active Shape 

This option allows users to remove or erase the active shape they have drawn on the chart. It provides users with the flexibility to correct mistakes or refine their selections without affecting other aspects of the chart or analysis. For example, if a user has drawn a shape to select a subset of data points but wants to modify or remove it, they can use the erase active shape option to remove it. 

 

[VIII] Zoom In/Out 

These options allow users to zoom in or out on specific areas of the chart to focus on particular details or to get a broader view of the data. 

 

 

[IX] Autoscale 

The autoscale option automatically adjusts the axes of the chart to fit the range of the data being displayed. It ensures that all data points are visible within the chart's boundaries, optimizing the visualization for better clarity and readability. Users can use this option to quickly adjust the scale of the chart without manually adjusting the axes, especially when dealing with datasets with varying ranges or outliers. 

 

[X] Reset Axes 

This option resets the axes of the chart to their default scales and positions. It provides users with a convenient way to revert any changes made to the axes, such as manual adjustments or zooming, and return the chart to its original state. Users can use this option to reset the chart for a fresh analysis or to restore the initial view after exploring specific regions or subsets of data. 

Save Layout

The Solutions Data Grid page allows you to save the current settings of your ad-hoc query for reuse. You can save the current view as a Layout and store the selected Properties, Objects, Phases, Periods, and more.

Layouts will save the time spent on selecting the same fields every time you evaluate a Solution.

  

 

Follow the steps below to save the Layout:

  2. Click on the grid icon located in the top right of the Solutions page.
  3. Using the Properties panel select the Objects and Properties that you want to view.

Drop Zeros from the Data Grid

We have provided a check box to hide values that are entirely zero for any of the Properties selected. This will not return the Property whose value is zero for the entire horizon. 

For example, let's assume you have a 7-day run generation with values of 0,0,0,0,0,0, and you clicked the checkbox, Drop Zeroes, then that property will not be returned in the response. However, if even a single day has a non-zero value (e.g., 0,0,0,0,2,0), the property will be displayed in the grid with all zeroes except for the one day with a value of 2.

To apply this in your output data grid page, check the box for Drop Zeros.

          

 

Query Multiple Samples and Time Slices from the Output Data Grid

 

You can  query the output for multiple Samples or Time Slices at once from the Output Data Grid page. To support this, the Output Data Grid page  has the option to select Samples or Time Slices in bulk. Once you click on APPLY, all the selected Samples or Time Slices will have their results displayed. Thus, saving the time and effort to query each time for different samples. This option is accessible with Monte Carlo and Auto Samples Simulation types. For single runs, the Sample option will be disabled and default to ‘Mean’, and Time Slice will default to ‘All Periods’.

 

NOTE: The Time Slice drop-down will only be displayed for Studies in which the Time Slice object is configured in the PLEXOS desktop.

 

To access multi-select Option –

 

1. Go to a Solution

2. Navigate to the Grid View

3. Select the required Objects and Properties from the Properties Panel and click APPLY.

4. For selecting multiple Samples, click the Sample drop-down.

5. Choose the Samples and click APPLY.
