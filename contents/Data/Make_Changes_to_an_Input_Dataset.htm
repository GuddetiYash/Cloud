Make Changes to an Input Dataset

 

Because PLEXOS Desktop is required to update input data, you must be an existing PLEXOS Desktop customer to complete these steps.

Currently, PLEXOS Cloud does not have features for changing an input dataset. Input data can be viewed but not changed. To make changes to an input dataset, follow these steps:

Overview

  1. On your local machine, open the project folder that contains the input data you want to change.
  2. Open the project’s XML file in PLEXOS Desktop.
  3. Make the desired data changes.
  4. Save the XML file.
  5. Upload the XML file to PLEXOS Cloud using PLEXOS Desktop or via CLI.

    For detailed steps on how to upload the Study from PLEXOS Desktop check  the article, PLEXOS Cloud Integration with PLEXOS Desktop , and from CLI check the article,  CLI Toolkit

  6. Return to PLEXOS Cloud. Your updated input data will be available in the Study drop-down menu.

 

Example: Update the Chile_Demo dataset

NOTE: If you do not have the Demo_Chile dataset, please request it from Energy Exemplar support.

  1. Open the Chile_Demo.zip file on your local machine.
  2. Then open the XML file in PLEXOS Desktop. Notice many of the properties are defined in CSV files.

  3. Make a simple data change that is easy to track. For this example, change the La_Silla_FV generator’s VO&M Charge from "0.1" to "2".



  4. Save the XML file.
  5. You can make any other data or settings changes in a similar manner. You can also adjust any values in the CSV files either in PLEXOS Desktop or externally in Excel/other software. Be sure to save the XML file.
  6. Upload the XML file to PLEXOS Cloud using PLEXOS Desktop or via CLI.

    For detailed steps on how to upload the Study from PLEXOS Desktop check  the article , PLEXOS Cloud Integration with PLEXOS Desktop and from CLI check the article, CLI Toolkit

 

View Input Properties

  1. Go to Solutions.
  2. Select a solution.
  3. On the Insights pane, open the Properties drop-down menu and turn on Generators.
  4. On the map, click on a generator. The Map & Bus Diagram will open.
  5. Click the Properties tab.

For more information about viewing data on the map, see the "Visualize Solution Data" section of Getting Started with PLEXOS Cloud.

 

View Advanced Configuration

  1. While you cannot currently adjust input data in PLEXOS Cloud, you can view Model settings by selecting Run >> Advanced Configuration

Edit Input Properties and view Connections for Map objects on the SETUP page

NOTE: This change is applicable for both Simulate and Analytics tier customers.


You can now edit the Input Properties for any object available on the map from the SETUP tab of the Data page. You do not have to Run the Simulation to view or edit the Input Properties and view Connections (Map or Bus diagram) for the selected object on the map. This feature saves time by allowing you to edit the Input Properties and view Connections on the SETUP page before running a Simulation. Previously, you could edit the Input Properties and View Connections for the selected map objects in the Map and Grid tab of the Solutions page.


NOTE: When you make changes to the Input Properties and click on the APPLY button, the new version of that Study gets created. You can view and manage the Study version in the VERSIONS tab.
