Overview

This article provides a guide on how to download and upload geocodes using the Setup tab in PLEXOS Cloud. Geocodes can be exported and imported in two supported formats: CSV and Shapefile. 

 

  • CSV (Comma-Separated Values): This is a plain text file format (which can be opened in Excel)  that contains details such as the Class Name, Object Name, and Geocoordinates (latitude and longitude) for each object.

  • Shapefile: A compressed ZIP file format commonly used in geographic information systems (GIS). It includes detailed polygon and spatial data for mapping and defining regions.

 

By understanding these formats, you can effectively manage your geocode data for analysis and visualization purposes.


Downloading Geocodes and Non-Geocodes

Downloading these files allows you to view, edit, and manage geocode/non-geocoded data in your preferred format for further analysis or updates.

 

To download geocode and non-geocode files from the Setup tab, perform the following steps: 

 

  1. Navigate to Data > SETUP.

  2. Click the DOWNLOAD button located at the top-right corner of the screen.

  3.  Select the file type (CSV or Shapefile) you wish to download. 

  4. The selected file will be downloaded to your system.

Uploading Geocodes and Non-Geocodes

Uploading functionality allows you to update geocode data directly in the Setup tab, ensuring the latest spatial information is reflected in your Model.

 

Follow these steps to upload geocode and non-geocode files to the Setup tab: 

 

  1. Navigate to Data > SETUP.

  2. Click the UPLOAD button located at the top-right corner of the screen.

  3. Select the file type (CSV or Shapefile) you wish to upload.

  4. Choose a file from your system or drag and drop the required file into the upload window.

  5. Click UPLOAD.
