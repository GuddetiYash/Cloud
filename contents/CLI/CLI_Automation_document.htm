PLEXOS Cloud CLI Automation

Global options


-f –format
Output format(json | csv | tsv | yaml | table | none)

 

-q --quiet
Turns off verbose messages from app

 

-aut --automation
Turns on developer friendly structured response messages from app
 

Auth Commands

Auth Login

The command below helps you log in to PLEXOS Cloud, using either SSO or Service Principal.


plexos-cloud auth login [-f | --format] [-q | --quiet] [-aut | --automation] [-c | --client-id] [-s | --client-secret] [-t | --tenant-id] [--service-principal]

 

Options


-c --client-id
Service Principal Client Id

 

-s --client-secret
Service Principal Client Secret

 

-t --tenant-id
Service Principal Tenant Id

 

--service-principal
Indicates that the credentials represent a service principal

 

Command Output

LoginResponse

 

Auth Logout

Use the command below to log out the current user and clear cached credentials.
 

      plexos-cloud auth logout [-f | --format] [-q | --quiet] [-aut | --automation]
