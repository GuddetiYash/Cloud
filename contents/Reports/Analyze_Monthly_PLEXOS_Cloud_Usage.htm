Analyze Monthly PLEXOS Cloud Usage

Stay ahead of the curve as a PLEXOS Cloud Admin with our latest Cloud Usage Summary feature. Each month, Cloud Admins will receive a PLEXOS Cloud Usage Summary email, which shows an in-depth analysis of the PLEXOS Cloud application's usage for their Tenant.  This email provides Admins  with crucial insights into the application's monthly activities, tailored separately for Credit Based and Core Based customers. Analyzing these details enables Admins to stay fully informed about what's happening within the platform to make informed decisions and take any necessary actions to optimize performance and efficiency .

The Admins will receive this email on the 3rd day of each month covering the PLEXOS Cloud usage summary for the previous month, spanning from the first to the last day. While Admins cannot change this schedule, they have the flexibility to manage this email notification under the Preferences > Email Preferences > Others section. 

By default, the toggle is enabled to receive monthly PLEXOS Cloud Usage email, but Admins can disable the toggle if needed.

 

Refer to the details of the email as mentioned below:

For Core Based Customers

  • Cloud Burst Usage (Consumed vs Remained)
  • Concurrent Cores Usage
  • Triggered Simulations Analysis
  • New Study Created and Deleted     
  • Total Allocated, Consumed, and Available Storage
  • Link to Release Notes, Knowledge Articles and Support Ticket

 

            

 

 

For Credit Based Customers

  • Compute Credits Usage
  • Concurrent Cores Usage
  • Triggered Simulations Analysis
  • New Study Created and Deleted     
  • Total Allocated, Consumed, and Available Storage
  • Link to Release Notes, Knowledge Articles and Support Ticket
     

            

 

 

Please  find the details of the entities in the Monthly Usage email for Core Based customers and Compute Credit Based customers as mentioned below:

  • SIMULATIONS- Shows the number of Simulations triggered for their tenant in the previous month, along with, Average Time, Total Time, Cores Used, and Peak Cores details. Additionally, a link to the Simulations page is provided, allowing users to easily manage their Simulations based on the analysis of the displayed data in this section.

  • STUDIES- Shows the number of Studies added, created (newly), and Deleted in the PLEXOS Cloud interface during the previous month. Additionally, a link to the Manage Storage page is provided, allowing users to easily manage their Storage based on the analysis of the displayed data in this section.

  • CLOUD BURST (For Core Based customers only)- Shows the total Cloud Burst Capacity, Days Used, and Days Remaining for their tenant. Additionally, a link to the Subscription page is provided, allowing users to easily manage their Cloud Burst usage based on the analysis of the displayed data in this section.

  • CREDITS (For Credits Based customers only)- Shows the total number of Purchased, Consumed, and Available Compute Credits for their tenant in the previous month. Additionally, a link to the Subscription page is provided, allowing users to easily manage their Credits utilization based on the analysis of the displayed data in this section.

  • STORAGE- Shows the total Allocated, Consumed, and Available space utilized by their tenant in the previous month. Additionally, a link to the Manage Storage page is provided, allowing users to easily manage their Storage based on the analysis of the displayed data in this section.

  • Support Ticket- Click on the link (support@energyexemplar.com) provided in the email, to initiate a conversation with Energy Exemplar Support Team via email for any additional details or requests.

  • Help Center and Release Notes- Click on these links to navigate to the PLEXOS Cloud Knowledge Base to explore the Articles, Release Notes, and much more to learn more about the product.


    NOTE: This applies to both Simulate and Analytics Tier customers.
