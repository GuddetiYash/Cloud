BI Analytics : How to create a chart with Secondary Y- Axis


A Secondary Y axis allows users to display multiple metrics or data series with different scales on the same chart, making it easier to compare and analyze data with disparate units or magnitudes. This makes it possible for users to display information from two different categories on a single chart or graph with the help of two new tables "data_secondary" and "fullkeyinfo_category_secondary," added to the Schema. Please refer to the example shown in the following screenshot and the steps below to understand the functionality.

 

 

The example above represents a line chart designed to provide a comprehensive comparison between two critical factors: "Generation," derived from the Generator class, and "Fuel Price," sourced from the Fuel Class. This chart utilizes a dual-axis approach, with "Generation" on the primary axis and "Fuel Price" on the secondary axis. This dual-axis design allows for a more nuanced and insightful analysis of the data, as it effectively showcases the relationship between these two variables in a single visualization.

 

NOTE: Please refer to the BI Schema Definition Doc  for the detailed definition of all the tables discussed below.

 

To build a similar visual of comparison between Generation and Fuel Price, follow the steps below:

 

1.    From My Reports Actions menu, click Edit to open  the BI Analytics editor, Select Line chart visualization from the Visualization pane.

 

 


 

2.    To set the Primary Y Axis value on the left side of the chart's Y axis, where the generated data will appear, please follow the steps below:     

 

        a.  From Data Pane’s “Fullkeyinfo_category”,     add field “childclassName”, open Filters, select Generator for   “childclassName”


 

       b.     Select Property Name from “Fullkeyinfo_category”, open Filters, select Generation as Property name.    


        c.  Add the corresponding value from the Data Table to the Primary Y-axis. You now have completed  defining the primary Y-axis.

  
3.    To set the Secondary Y Axis value on the left side of the chart's Y axis, where the fuel data will appear, please follow the steps below:  

 

    a.  Add childclassName_Secondary from "Fullkeyinfo_category_Secondary", open Filters,  select Fuel as   “childclassName”


         b.   Add the PropertyName_Secondary from "Fullkeyinfo_category_Secondary", open Filters  and select  Price  as Property name.


         c.   To set the X Axis as Date, Drag and drop Year to ‘X’ axis from "Period_Start_Date"   (Fullkeyinfo_category_Secondary ).


         d.   Add corresponding value from "data_Secondary" table to Secondary Y-axis.

 

5.  You can now see the compared line chart visualization between Generation and Fuel Price as shown in the above image. 
You  can further customize your chart, add data labels, legends, and more to make it more informative and visually appealing.


NOTE: Currently, the use of secondary classes is not supported with membership tables. Other Bi features will be supported as it is.
