Mapping Cloud to Desktop Terminology

This article maps the differences in PLEXOS Desktop and Cloud terminology.
 

Desktop Cloud Description
Dataset Study A collection of models stored in a single XML file
Diagnostic files Artifacts Log files, lp files, solution files, and other help files associated with the model
Visualizer Setup This allows you to view the map and geocoded version of your model setups
Execute Run The command to run/execute your model with the PLEXOS engine
Execute Apply When querying your solution data, in Desktop you Execute the query. In Cloud, you click Apply to view the selected property values on the Solution tab.
