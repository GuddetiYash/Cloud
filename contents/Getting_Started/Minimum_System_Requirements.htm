Minimum System Requirements

Introduction

This System Requirements Document for PLEXOS product is a comprehensive guide outlining the minimum hardware, software, and network specifications necessary to use the PLEXOS product effectively. PLEXOS Cloud is a cloud-based energy market simulation and optimization platform designed for the energy domain. 

 

Hardware Requirements

Below are the minimum hardware configurations needed to run the PLEXOS product effectively:

 

  • Processor: 8th Generation Intel® Core i5-8350U Processor (Core i5-8350U)

  • RAM: 8GB DDR4-2400

  • Storage: 256 GB   

    NOTE: The above-mentioned Hardware Requirements are specifically for PLEXOS Desktop. 


Learn more about the System Requirements for the PLEXOS Desktop and Aurora Desktop.

 

Software Requirements

Below are the supported web browsers for optimal user experience with PLEXOS Cloud:

 

  • Chrome: Version 83 and above

  • Edge: Version 83 and above

  • .NET Core/Framework: Version 6.0 and above 

    NOTE: .NET Core applies to versions 5.X and later, and .NET Framework applies to versions 4.X and earlier.

  • Safari: Not supported

  • Internet Explorer: Not supported

  • Firefox: Not Supported

Network Bandwidth Requirements

Below are the Network Requirements for accessing PLEXOS Cloud and utilizing the Command Line Interface (CLI) and the Microsoft Excel Add-In:

 

  • Ingress and Egress Port Configuration: Enable both the Inbound (ingress) and outbound (egress) traffic on port 443 for HTTPS to ensure that the network configuration permits inbound and outbound traffic on port 443 for HTTPS.

  • Whitelisting Domain: Whitelist *.energyexemplar.com to allow access to any subdomain under energyexemplar.com.

  • Handling a Company Proxy (if Applicable): If your organization uses a company proxy, configure it to permit traffic to and from *.energyexemplar.com on port 443.  

    Learn more about Proxy Settings for using PLEXOS Cloud.

 

Known Limitations

Known Limitations for Pushing Data from PLEXOS Desktop to PLEXOS Cloud  are:

 

  • Creating/Adding New Study: Uncompressed study size of up to 12 GB is supported. 

  • Creating/uploading a new version:

    • XML Changes: 500,000   records (there should not be more than 500,000 changes in the XML. By changes, includes add, removes & modifies.)

    • CSV/TimeSeries: No more than 3 GB of total timeseries changes can be pushed. For example, if 5 files change, the sum of their change (diff) size should not be more than 3 GB.
