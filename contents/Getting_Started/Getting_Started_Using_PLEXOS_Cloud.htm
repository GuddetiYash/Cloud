8. PLEXOS Cloud Integration with PLEXOS Desktop

Starting from PLEXOS Desktop version 9.1 R03, a new Cloud Integration capability has been introduced, enabling a seamless connection between PLEXOS Desktop and PLEXOS Cloud. This integration streamlines key workflows by allowing you to access Cloud functionality directly within the PLEXOS Desktop environment, minimizing the need to switch between applications.

Â 

PLEXOS Desktop version 9.1 R03 and above includes the following Cloud-integrated features:

Â 

  1. Save a Study to PLEXOS Cloud: Upload your Study from PLEXOS Desktop to the Cloud to take advantage of scalable computing and collaboration.

  2. View a Study in PLEXOS Cloud: Launch and explore the selected Study directly in the Cloud environment.

  3. View the Solution list and Download\view Solutions in PLEXOS Cloud: Access your Solution list from the Cloud and download or analyze them as needed.

  4. Run and Configure a Model\View Model in PLEXOS Cloud: Trigger and configure simulation runs in the Cloud from the Desktop interface, enabling quicker access to advanced configurations.

  5. Sync a Study from Desktop to PLEXOS Cloud and vice versa: Synchronize Studies between your local Desktop and the Cloud environment. You can push updates from Desktop to Cloud or pull changes from the Cloud to your local system.

1. Credit Calculation method

The Credit Calculation method for Simulation Runs uses a 5-minute interval block structure. Credits are deducted in 5-minute blocks throughout the entire Simulation, enhancing Credit utilization efficiency. This method is especially beneficial for Models with shorter execution times, allowing you to Run these Models using fewer Credits.

Â 

[A] How It Works

Refer to the example and formula below to understand the 5- minute interval Credit Calculation Method:
Â 
If the Simulation Run time is less than 5 minutes on a 2 Core Machine, the system will deduct 0.17 Credits for that Simulation. Similarly, if the Simulation Run time is 10 minutes on a 2 Core Machine, the system will deduct 0.17 Credits for the first 5 minutes and continue adding for each subsequent 5-minute interval until the Simulation is completed. For instance, for a 10-minute Simulation, the system will deduct approximately 0.34 Credits (0.17 +0.17). Â Â
