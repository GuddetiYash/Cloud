Simulate and Analytics Bundle in PLEXOS Cloud

PLEXOS Cloud Bundles and Licensing

To address the diverse challenges encountered by users of PLEXOS Cloud, we have implemented a set of flexible licensing options, presented in specially curated Bundles designed to meet users' unique Simulation requirements. 
The introduction of these two new Bundles, known as Simulate and Analytics, offers users distinct options with tailored features and capabilities to enhance their PLEXOS Cloud experience.


Simulate Bundle:

The Simulate Bundle is designed to offer scalable and flexible computing capabilities for executing Simulations in PLEXOS Cloud, along with robust version management and collaboration features. Analysts utilize the cloud infrastructure to run Simulations and utilize the desktop application for analyzing the output.

 

NOTE: The feature table at the bottom of this article describes the list of functionalities available with the Simulate Bundle.

 

Analytics Bundle:

The Analytics Bundle is designed to provide the users with enhanced features and functions for a deeper understanding of their data. With the Analytics Bundle, users gain access to powerful output analysis tools including Map Visualizations, ad hoc queries, BI Reports, the Excel-AddIn, powerful BI Analytics capabilities, Event Detection for advanced Reporting and dashboarding in PLEXOS Cloud.

The Analytics bundle is priced per named user (Analyst). This gives all named user the access to additional features and functions including embedded Microsoft Power BI for Reporting/dashboarding.

Power BI Reporting is only available with PLEXOS Cloud and cannot be added to stand-alone PLEXOS Desktop.

Both Simulate and Analytics Bundle provides the flexibility to choose between "Core Based Pricing" or "Usage Based Pricing" for its pricing model. 

 

  • Core Based (Base + Cloud Burst):  Core Based offers continuous availability, ensuring that users have a minimum amount of computational power throughout the contract term.

Additionally, for those seeking additional compute capacity, Cloud Burst Cores are the answer. They provide an on-demand solution, available in 24-hour increments, which supplements the base cores when user needs a burst of computational power. Thirty days of Cloud Burst Cores are included for all Core Based Pricing tenants.

This option is tailor-made for teams with steady Simulation demands and minimal variability or scalability needs.

 

  • Usage Based (Credits): Usage-Based offers several benefits. Here Costs are determined per credit, applied during Simulation runs. It helps teams and consultants accurately calculate project costs, enables seamless scalability to meet demand, and provides quick access to Simulations. This means users can efficiently carry out their tasks without unnecessary delays.
